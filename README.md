#  Bug Simulator

Learn React by fixing real-world bugs.

Bug Simulator is an interactive platform where developers can practice debugging React applications through realistic coding challenges.

---

##  Features

- Interactive bug-fixing challenges
- Multiple difficulty levels
- Real React mistakes and edge cases
- Progress tracking
- Score system
- Modern UI

---

##  Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS
- React Router

##  Why this project?

Most developers learn React by building projects.

Bug Simulator helps developers improve a different skill:

> Finding and fixing bugs in existing code.

This is one of the most valuable skills for real-world frontend development.
